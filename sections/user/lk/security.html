Защита по IP адресу
  • Защита: |!security_ip|выключена|_!security_ip| |security_ip|включена|_security_ip|
    |!security_ip|
    Включить
    |_!security_ip| |security_ip|
    Выключить
    |_security_ip|
Если включена защита по IP адресу, то войти в панель управления можно только с указанных одиночных ip или подсетей.

Доверенные адреса и подсети:

    [security]

Ваш ip адрес: [ip]

Ваша подсеть: [subnetwork]

Защита по коду
  • Защита: |!security_code|выключена|_!security_code| |security_code|включена|_security_code|
    |security_code|
    Выключить
    |_security_code| |!security_code|
    Включить
    |_!security_code|
    • Если включена защита по коду, то при каждой авторизации нужно вводить код, который будет отправлен на почту.
